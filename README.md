# Mini Task Manager

A full-stack task management application built with **Next.js (App Router), TypeScript, Node.js, Express, MongoDB, and Tailwind CSS**.

## Status

🚧 Project currently under development.

The application will support:

* Add a task
* View all tasks
* Mark a task as completed
* Delete a task

The project is being developed as part of a technical assessment, with a focus on clean architecture, input validation, error handling, and maintainable code.
